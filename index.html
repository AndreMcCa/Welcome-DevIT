=== GIT внесение изменений ===

Рабочее окружение

Вы уже работаете на ветке task_4 родительская ветка которой dev.

Задание

На ветке dev удаленного репозитория появились новые изменения и ваша задача добавить их в свою ветку перед своими, опишите порядок ваших действий.

Мы находимся в ветке task_4. Для того, чтобы стянуть последние изменения, нужно перейти в ветку dev с помощью команды git checkout dev и выполнить команду git pull. После снова вернуться в ветку task_4 командой git checkout task_4 и синхронизируемся с веткой dev, при помощи комманды git merge dev

Результат

Если ветка dev только моя и над ней работаем только я, тогда можно использовать команду git rebase dev

...после перейти в ветку dev => git checkout dev, и выполнить git rebase task_4.
